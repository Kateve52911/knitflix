# Code of Conduct

This project aims to foster an open and welcoming environment. To ensure this, I ask all participants to:

## Be Respectful

- Use inclusive language and respect diverse perspectives
- Accept constructive criticism gracefully
- Focus on what's best for the community and project
- Show empathy towards other community members

## Be Helpful

- Clearly document issues and feature requests
- Provide context and relevant details when reporting bugs
- When contributing code, include appropriate tests and documentation
- Help others who are less experienced when possible

## Unacceptable Behavior

- Harassment, discriminatory jokes/comments, or personal attacks
- Trolling or derogatory comments
- Publishing others' private information without permission
- Any conduct reasonably considered inappropriate in a professional setting

## Enforcement

Issues can be reported by contacting the project maintainer. All complaints will be reviewed and investigated promptly and fairly.

Contributors who do not follow this Code of Conduct may face temporary or permanent repercussions as determined by the project's leadership.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org).
